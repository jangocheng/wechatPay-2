# wechatPay
微信支付

配置自己的支付参数
AppId = ""          //公众号appid

AppSecret = ""      //公众号秘钥

MerId = ""          //商户号

PayKey = ""         //商户秘钥

NotifyUrl = ""      //公众号支付回调

SpbillCreateIp = "" //发起请求的机器ip

NativeUrl = ""      //扫码支付回调


JsApiPay 获取公众号支付信息（生成预支付订单）

JsApiPayCallBack 公众号支付回调

NativePay 回去扫码支付二维码

NativeCallback 扫码支付回调
